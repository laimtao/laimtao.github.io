liunx常见命令：

cd:改变目录

cd..:返回上一级

pwd:显示当前所在目录路径

ls:显示当前目录所有文件

touch:新建一个文件

rm:删除一个文件

mkdir:删除一个文件夹

mv:移动文件

clear:清屏

exit:退出

top:查看cpu和内存

git：

git init:初始化仓库

git add:添加文件到暂存区

git commit:将暂存区内容添加到仓库中

git pull:下载远程代码并合并

git push:上传远程代码并合并

git checkout:切换分支

git merge:合并分支

git clone:将远程服务器上的仓库完全镜像一份到本地

git工作区域：

workspace：工作区，平时存放代码的地方

index/stage：暂存去，临时存放改动

repository：本地仓库，存放数据的位置，存放着提交到所有版本的数据，其中head指向最新版本

remote：远程仓库，托管代码的服务器
